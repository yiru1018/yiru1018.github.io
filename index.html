個人簡介

我是施怡如，一個從學生時期就對護理沒有興趣的護理師。 畢業後，先後任職於兒科診所、醫美診所，雖然薪水穩定，但診所護理師的工作真的沒有什麼挑戰，漸漸感到無趣。在醫美診所時我遇到了促使我轉職的關鍵人物——帶我的診所學姊。 看著他工作時，完全可以感受到他對工作的熱愛。他跟我分享了他為了進修而導致睡眠不足 ，但是他看起來是這麼的甘之如飴。我當時心想，原來真的有熱愛工作的人。是他， 讓我決定我要追求我自己真正想做的事情。我想繼續朝軟體工程師發展，我想變得更厲害， 想成為像學姊那種工作時閃閃發光的人。

為什麼想成為前端、後端、或全端工程師？

剛開始轉職時，蒐集了大量的資訊，也詢問過別人的意見。大部分的人都推薦非本科系或文組者想轉職的話，前端是最好入門的。相較於後端比較不會需要這麼多的數學知識，而且資源很多較易自學。由於我身邊並沒有從事這方面工作的人，且非本科系，所以想從前端先入行，之後看情況做調整，因為我本身也沒有排斥後端。

為了成為軟體工程師，曾經做過什麼努力？

  1. 參加了工研院的 Python 商業資料科學人才養成班。有了對程式語言基本的認識，也發現我蠻喜歡寫程式的。課程結束前也獨自完成了一個小小的專題。
    血糖監測器 ：記錄血糖的GUI，將數值圖表化，方便監測。
  2. udemy課程進度
  3. 自學筆記
  4. 最近開始加入medium筆記

如果參與這個計畫，會怎麼安排學習時間？

會把兼職的工作都辭掉改為全職學習，一到六一天安排八小時完成課堂任務，若提早完成可額外學習相關知識。（例：之前參加python課程時若完成老師給的作業還有時間的話，我會嘗試看看自己能不能用更好的辦法寫出來，或自己擴充別的功能）星期日是情況做調整看是要休息還是繼續練習。

如果參與這個計畫，預期會碰到什麼困難？你打算怎麼解決它？

寫程式一定會有卡關的時候，這時候google是我的好朋友，stackoverflow也有很多資訊可以參考。真的都找不到的時候我會問同學或老師，但是老師多半給我的答案是我也不知道你可能要google一下，所以會再重回google的懷抱。其實根據我之前的經驗，真的大多數我想要的東西都找得到，所以找不到的時候就重下關鍵字，還是找不到，就想替代方案，用別的方式達到相同的效果。

是否有想要加入的軟體公司？為什麼想加入該公司？

希望有機會的話可以在google工作，理由如下：

  1. 首屈一指的公司裡一定有許多強者，若跟這些人一起工作可以從他們身上學到很多東西。
  2. 想體驗跟各國人士工作的感覺。
  3. 看了許多人分享google的面試流程，層層關卡，若可以通過，對我來說是一種很大的肯定。
  4. 想實驗看看這種知名外商是否也是學歷至上，若我這非本科學歷也普通的人都可以通過的話，就可以鼓勵更多跟我一樣背景但是想轉職的人。
  5. 一個職涯上的終極目標，激勵自己不斷往前。

想要對我們說的事情？

一直以來，我都不是個擅長理科的人，可以說我就是個文組腦。學生時期略略接觸過程式，但純粹 是因為我覺得這是世界趨勢，科技會不斷進步，連小學生都在學，我也要懂一點，後面並沒有做更深入的學習。 直到參加了python的課程，才發現自己是喜歡的，人生中也從來沒想過會想當一個工程師。可是我不想讓科系以及擅不擅長 這件事把自己困住，因為完成作品的成就感，真的是太爽了。我想要學更多讓自己變得更厲害，也想補足身為軟體工程師該有的 基礎知識不管是演算法還是資料結構，總之我想盡一切可能讓自己可以把這件事做得更好。彭彭老師要求的網頁我加了一點javascript 的部分，是之前線上課程剛好有學到我想說可以順便練習。我知道大家都是抱持得相當的決心想加入這個課程，自己並無特別亮眼之處。 但我想要讓自己變好的念頭是不會輸給任何人的。
